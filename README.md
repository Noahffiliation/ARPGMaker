# ARPGMaker
ARPG Game Engine for COS370

## Features
- Map
  - Adjustable tile system
    - Background loading
  - Camera
    - Displays dimensions for number of tiles
- Entity class
  - Player, NPCs/Enemies
  - Animations (Basic)
- Rendering
  - Render layers of ground, textures, players/buildings on each tile
- GUI
  - Menu helper functions
- Audio
